# Muma Kalobwe - Portfolio Website

A modern, responsive portfolio website showcasing the work of Muma Kalobwe, a full-stack developer specializing in FinTech, AI, and data-driven solutions.

## 🚀 Features

- **Responsive Design**: Mobile-first approach with seamless desktop experience
- **Dark/Light Mode**: Toggle between themes with smooth transitions
- **Interactive Animations**: Smooth scroll-triggered animations using Framer Motion
- **Project Showcase**: 15+ comprehensive projects with live demos and documentation links
- **Tech Stack Display**: Visual representation of technical skills and tools
- **Educational Background**: Timeline of academic achievements
- **Contact Integration**: Direct email contact functionality

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: Tailwind CSS, Custom CSS Variables
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **UI Components**: Radix UI primitives
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/BlvckKryptonite/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🎨 Design System

- **Typography**: Poppins for headings, Inter for body text
- **Color Scheme**: Blue gradients (#3B82F6 to #6366F1) with slate variations
- **Animation**: Smooth transitions with Framer Motion
- **Responsive**: Mobile-first design with Tailwind breakpoints

## 📱 Responsive Design

The website is optimized for all screen sizes:
- Mobile: 320px+
- Tablet: 768px+
- Desktop: 1024px+

## 🚀 Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions. Every push to the main branch triggers a new deployment.
